<h1>Akiya Scraper</h1>

<h2>Description</h2>
A web scraper used to get information on various akiyas(empty houses) and properties across Japan.
<br />


<h2>Language(s) Used</h2>

- <b>Go</b> 

<h2>Environments Used </h2>

- <b>Unbuntu Server 22.04 LTS</b>
- <b>VirtualBox</b>
